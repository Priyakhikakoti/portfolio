# Priyakhi Kakoti - Personal Portfolio Website

A premium, fully responsive, and highly interactive personal portfolio website showcasing my technical skills, internship experience, projects, and achievements.

🎨 **Live Demo:** [https://Priyakhikakoti.github.io/portfolio/](https://Priyakhikakoti.github.io/portfolio/) 

---

## ✨ Features

- **Glassmorphism Aesthetic:** Modern, sleek interface styled with customized HSL variables, smooth gradients, and backdrop-filters.
- **Adaptive Light/Dark Theme:** Fully toggleable light/dark mode that persists across sessions via local storage.
- **Fluid Micro-Animations:** Dynamic circular skills indicators, typewriter introduction, hover-based card expansions, and custom timeline reveals.
- **Categorized Project Filter:** Easily sort projects by Full Stack, ML & Data Science, Backend & Python, and Frontend categories.
- **Interactive Modals:** Custom card views displaying in-depth specifications for projects and structured mockup certificates for credentials.
- **Mobile First & Responsive:** Optimized to load quickly and display beautifully on all screen resolutions (Ultra-wide, Desktop, Tablets, and Mobile).

---

## 🛠️ Built With

- **Structure:** Semantic HTML5
- **Styling:** Custom Vanilla CSS3 (Grid, Flexbox, Keyframes)
- **Interactivity:** Vanilla JavaScript (ES6)
- **Icons:** Bootstrap Icons CDN

## 🚀 How to Run Locally

1. Clone this repository:
   ```bash
   git clone https://github.com/Priyakhikakoti/portfolio.git
   ```
2. Open `index.html` directly in any web browser, or serve it using a local server:
   ```bash
   # If Python is installed
   python -m http.server 8000
   ```
3. Navigate to `http://localhost:8000` in your browser.
